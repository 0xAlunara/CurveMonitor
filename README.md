# CurveMonitor

1. Clone the repo
   ```sh
   git clone https://github.com/phil-svg/CurveMonitor.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Launch backend via
   ```sh
   node testing_CurveMonitor.mjs
   ```
4. Launch frontend via
   ```sh
   node socketHandlerFrontend.mjs
   ```
